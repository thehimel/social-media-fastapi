## Course Overview

**Sanjeev Thiyagarajan** will be the instructor for this course. The course teaches how to build **APIs** in **Python** using **FastAPI**, a popular framework that makes it easy to build highly performant **APIs**.

### What You Will Learn

**API Development**

* Common conventions when building **APIs**
* The ins and outs of **FastAPI**
* Reading and writing data to a **PostgreSQL** database

**Authentication**

* Implementing **authentication** within the **API**
* User sign up and log in using **JSON Web Tokens** (JWT)

**Testing and Deployment**

* Testing **APIs** using **Pytest**
* Handling **database migrations**
* Deploying applications to two platforms: **Ubuntu** server and **Heroku**
* **Dockerizing** applications to run within **Docker** containers
* Building a **CI/CD pipeline** to automate testing and deployment

### Outcomes

By the end of the course, you will have a solid understanding of how to:

* Build robust **APIs**
* Properly test and deploy them

Whether you are building a back end for a web app or creating an **API** for a machine learning model, you will have the tools needed to get up and running.
